Requisitos

1. Windows 10
2. Python 3
3. Pip3
4. Biblioteca CV

Instalacao

Dentro da Microsoft Store obtenha o python 3.7, e após isso execute o comando "pip install Pillow" dentro de um terminal, caso isso não funcione siga os seguintes passos:

1. Baixe e instale o python 3: https://www.python.org/ftp/python/3.7.9/python-3.7.9-amd64.exe (64 bits) ou https://www.python.org/ftp/python/3.7.9/python-3.7.9.exe (32 bits)
2. Adicione o pithon nas váriaveis de sistema do windows
3. Para instalar o pip3:
	- Baixe o arquivo https://bootstrap.pypa.io/get-pip.py
	- Execute o arquivo com python get-pip.py
4. Dentro de um terminar execute "pip install opencv-python" (ou pip3 install opencv-python caso tenha duas versões do Python  na maquina)

Intruções

1. Coloque todas as imagens necessarias na pasta 'imagens/folhas' (algumas imagens de teste já estão lá)
2. Execute o arquivo main.py com o comando: python main.py (python3 main.py caso tenha as duas versões do python instaladas)
3. Insira o caminho da pasta com as imagens, relativo a pasta onde o projeto está
3. Insira o nome da imagem sem sua extensao
4. Entre com o tamanho raio do Segmento da Cadeia de Inclinação (e.g:1, 10, 100)
5. Aguarde o programa terminar a execução
6. O resultado pode ser encontrado na pasta das imagens
