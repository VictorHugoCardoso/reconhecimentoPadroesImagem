Requisitos

1. Windows 10
2. Python 3
3. Pip3
4. Biblioteca Pillow

Instalacao

Dentro da Microsoft Store obtenha o python 3.7, e após isso execute o comando "pip install Pillow" dentro de um terminal, caso isso não funcione siga os seguintes passos:

1. Baixe e instale o python 3: https://www.python.org/ftp/python/3.7.9/python-3.7.9-amd64.exe (64 bits) ou https://www.python.org/ftp/python/3.7.9/python-3.7.9.exe (32 bits)
2. Adicione o pithon nas váriaveis de sistema do windows
3. Para instalar o pip3:
	- Baixe o arquivo https://bootstrap.pypa.io/get-pip.py
	- Execute o arquivo com python get-pip.py
4. Dentro de um terminar execute "pip install Pillow" (ou pip3 install Pillow caso tenha duas versões do Python  na maquina)

Intruções

1. Coloque todas as imagens necessarias na pasta 'imagens' (algumas imagens de teste já estão lá)
2. Execute o arquivo main.py com o comando: python main.py (python3 main.py caso tenha as duas versões do python instaladas), uma aba do navegador será aberta
3. Insira o nome da imagem com sua extensao no campo "Nome da imagem" (a imagem deve estar na pasta 'imagens')
4. Selecione quais algoritmos serão executados marcando o checkbox de cada um
5. Insira a quantidade de vezes que o algoritmo será executa no campo "Repeticoes" ao lado do checkbox do algoritmo
6. Clique na opcao enviar e aguarde o programa finalizar a execucao, as imagens serão geradas dentro da pasta 'imagens/geradas/'
